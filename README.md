# BusyBus

### Description
BusyBus is a transit system mobile application that provides an easy way to understand local city bus routes. This project shows how the site looks when viewing one specific bus stop. 

### Built with
This prototype was used using HTML5 & CSS3 following Material Design guidelines. Layout was made using CSS Grid.

### Getting Started

Clone this directory via this command. 

```bash
git clone https://github.com/jessbradford/BusyBus 
```

This will create a folder in the current working directory called BusyBus. Change into the directory with the following command.

```bash
cd BusyBus
```

Make any changes using your favorite editor and open index.html to see changes.
